## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/kanugurajesh/Portfolio.git
   ```
2. **Install the dependencies:**
   ```bash
   cd Portfolio
   npm install
   ```
3. **Run the Project:**
   ```bash
   npm run dev
   ```
